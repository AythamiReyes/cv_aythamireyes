SistemaBibliotecaMVC

Descripcion breve sobre el proyecto: En este proyecto creamos un sistema de biblioteca con MVC, utilizando clases bajo el patrón MVC, el sistema controla y permite gestionar libros y usuarios, prestamos y devoluciones, implementa metodos para manipular los datos y mostrarlos de manera estructurada aplicando como en el proyecto anterior, clases, objetos, enum, atributos y validaciones. Java

SistemaPersonajesVideojuego

Proyecto para desarrollar la lógica de un sistema de personajes de videojuego, con sus atributos, los personajes comparten características comunes pero cada personaje actúa de forma distinta(ataque, habilidades, capacidades). En la jerarquía MVC, tenemos una clase abstranta personaje la cual es obligatoria, que a su vez tiene otras dos subclases personajefisico y personajemagico, con al menos algun metodo no abstracto como por ejemplo cargarAtaque. Mímino 3 interfaces obligatorias, clase gestora del juego, Main para ejecutar y mostrar, y consola para imprimir y leer.

Java

GestionBancos.

El programa define una clase que representa una cuenta bancaria con: Titular Saldo Métodos para: Ingresar dinero Retirar dinero (con control de saldo insuficiente) Transferir dinero entre cuentas Consultar el saldo actual La clase crea dos cuentas y ejecuta un ciclo de 5 operaciones aleatorias. Dependiendo de la operación seleccionada, el programa: Muestra el saldo de ambas cuentas Solicita al usuario una cantidad para ingresar Solicita una cantidad para retirar Solicita una cantidad para transferir de una cuenta a otra Cada operación muestra por consola el resultado y el nuevo estado de las cuentas.
